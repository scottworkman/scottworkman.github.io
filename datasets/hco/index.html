Dataset Details

Technical specifications

The HoliCity-Overhead (HCO) dataset was introduced to support the new task of geo-enabled depth estimation, which leverages geospatial context to resolve the scale ambiguity inherent in monocular vision. As an extension of HoliCity, the dataset was used to illustrate that when a camera’s location is known, co-located overhead imagery can be used to anchor a scene’s scale—overcoming the sparsity and long-range precision limitations of active sensors like LiDAR. HCO provides a city-scale framework for fusing ground-level panoramas with overhead imagery and digital surface models (DSMs) to achieve absolute metric depth estimates, significantly reducing errors at large distances—up to several kilometers.

Dataset Features

City-Scale Extent

Covers over 20 km² of downtown London, providing dense urban structures and diverse architectural styles for large-scale evaluation.

Multi-Modal context

Pairs registered ground-level images with overhead satellite imagery and high-resolution digital surface models (DSM).

Metric Depth Labels
